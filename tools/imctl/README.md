# imctl

Command Center CLI for Infinite Money.

Commands:

- `imctl doctor`
- `imctl backtest`
- `imctl optimize`
- `imctl report`
- `imctl knowledge sync`
- `imctl checks`
- `imctl data qc-market --zip data/equity/usa/daily/spy.zip --symbol SPY --out data/market_data.csv`
- `imctl signals export --config hedge_fund/conf/ai_stack.yaml`
- `imctl sr report --config hedge_fund/conf/ai_stack_csv_sr_barrier_demo.yaml`
- `imctl sr sweep --config hedge_fund/conf/ai_stack_csv_sr_barrier_demo.yaml --n-trials 30`
- `imctl sr backtest --config artifacts/<run_id>/tuned_config.yaml`
